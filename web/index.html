About the project

Smart Home is a project that was conducted by students studying Computer Science at Kristianstad University. The idea was for a platform to control a whole house using a simple and innovative application.

Different technologies were used. All of the technologies are listed underneath.

For the clouding we used Azure Microsoft SQL Server 201, Azure Node.js REST API and PubNub.

For the hardware we used Raspberry pi for the home server and using ZigBee for communication. We also used Arduino micro controllers which also use ZigBee for communication.

The units we are using are Smart Phones using Android and a web application which we can use a browser to access.

Different programming languages and scripts were used. We used Python for the Raspberry pi units. Processing was used with the arduino units. We used smart phones that have the Android operating system installed . The browser based client uses HTML, CSS, PHP, JavaScript and Bootstrap for the front-end. The API we are using is Node.js and the database is SQL.
